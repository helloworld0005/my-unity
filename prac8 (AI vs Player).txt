Practical 8
Aim: 1. AI vs Player
________________


Step 1: Create a 3D Project
* Open Unity and create a new 3D project.
* The Unity editor will open with a default SampleScene containing a Main Camera, Directional Light, and Global Volume.
Step 2: Create the Environment
* In the Hierarchy window, right-click and select 3D Object > Plane.
* Select the Plane and in the Inspector, set its Scale to:
   * X: 2.03
   * Y: 1.07
   * Z: 6.63
* Output Description: A flat, elongated horizontal surface appears in the center of the scene.
Step 3: Add Color to the Plane
* Go to the Project window at the bottom and navigate to the Scenes folder.
* Right-click on the folder, select Create > Material.
* In the Inspector for the new material, click on the Base Map color picker.
* Choose a color (e.g., green with Hexadecimal: 6CDEAC) from the Color window and close it.
* Drag and drop this material from the Assets folder directly onto the Plane in the Scene view.
* Output Description: The white plane is now filled with the selected green color.
Step 4: Create Game Objects (Goal, Player, and Enemy)
* Goal: In the Hierarchy, right-click > 3D Object > Cube. Rename it to Goal.
   * Transform Position: X: 10, Y: 0.5, Z: 0.
   * Transform Scale: X: 3.09, Y: 4.08, Z: 1.
* Player: Right-click > 3D Object > Capsule. Rename it to Player.
   * Transform Position: X: 1.67, Y: 1, Z: -6.4.
   * Transform Scale: X: 0.41, Y: 0.5, Z: 0.05.
* Enemy: Right-click > 3D Object > Capsule. Rename it to Enemy.
   * Transform Position: X: -0.06, Y: -7.36, Z: 0.
   * Transform Scale: X: -0.56, Y: 0.85, Z: 1.
Step 5: Configure Physics and Components
* Goal:
   * Click Add Component > Physics > Box Collider.
   * Check the Is Trigger box.
* Player:
   * Click Add Component > Physics > Rigidbody.
   * Uncheck Use Gravity.
   * In Constraints, check Freeze Rotation for X, Y, and Z.
   * Go to the top of the Inspector, click the Tag dropdown, and select Player.
* Enemy:
   * Click Add Component > search "nav" > select Nav Mesh Agent.
   * Set Base Offset to 1.
   * Click the Tag dropdown > Add Tag... > click + > type "Enemy" > Save.
   * Go back to the Enemy and select the newly created Enemy tag.
Step 6: Bake the NavMesh
* In the Hierarchy, right-click > Create Empty. Name it navmesh.
* In its Inspector, click Add Component > NavMesh Surface.
* Click the Bake button at the bottom of the component.
* Output Description: A blue semi-transparent overlay appears on top of the Plane in the Scene view, indicating the walkable area.
Step 7: Scripting
Player Movement (player.cs)
Create this script in Assets > Scenes.
C#
using UnityEngine;


public class player : MonoBehaviour
{
    public float speed = 5f;
    Rigidbody rb;


    void Start()
    {
        rb = GetComponent<Rigidbody>();
    }


    void FixedUpdate()
    {
        float h = Input.GetAxis("Horizontal");
        float v = Input.GetAxis("Vertical");
        Vector3 move = new Vector3(h, 0, v) * speed;
        rb.linearVelocity = new Vector3(move.x, rb.linearVelocity.y, move.z);
    }
}


* Action: Attach this script to the Player object.
AI Behavior (AI_Players.cs)
Create this script and attach it to the Enemy.
C#
using UnityEngine;
using UnityEngine.AI; // Ensure this is added for NavMesh


public class AI_Players : MonoBehaviour
{
    public Transform target;
    public float speed = 3.5f;
    private NavMeshAgent agent;


    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
        agent.speed = speed;
    }


    void Update()
    {
        if (target != null)
        {
            agent.SetDestination(target.position);
        }
    }
}


* Action: In the Enemy Inspector, drag the Goal object into the Target slot of this script.
Goal Logic (goal.cs)
Create this script and attach it to the Goal.
C#
using UnityEngine;


public class goal : MonoBehaviour
{
    private void OnTriggerEnter(Collider other)
    {
        if (other.CompareTag("Player"))
        {
            Debug.Log("Player wins");
            Time.timeScale = 0;
        }
        else if (other.CompareTag("Enemy"))
        {
            Debug.Log("AI Win");
            Time.timeScale = 0;
        }
    }
}


Game Manager (GameManager.cs)
Create a new Empty Object in Hierarchy called GameManger and attach this script.
C#
using UnityEngine;


public class GameManager : MonoBehaviour
{
    public GameObject player;
    public GameObject ai;
    public GameObject goal;


    void Update()
    {
        float playerDistance = Vector3.Distance(player.transform.position, goal.transform.position);
        float aiDistance = Vector3.Distance(ai.transform.position, goal.transform.position);


        if (playerDistance < 2.0f)
        {
            Debug.Log("Player Wins");
            QuitGame();
        }
        else if (aiDistance < 2.0f)
        {
            Debug.Log("AI Wins");
            QuitGame();
        }
    }


    void QuitGame()
    {
        Application.Quit();
        UnityEditor.EditorApplication.isPlaying = false;
    }
}


* Action: Drag the Player, Enemy, and Goal objects from the Hierarchy into their respective slots in the GameManger component.
Step 8: Final Run
* Press the Play button.
* Use arrow keys to move the Player toward the Goal.
* Output Description:
   * The Enemy capsule moves automatically toward the Goal.
   * If the Enemy reaches first, the Console prints "AI Wins" and the game stops.
   * If the Player reaches first, the Console prints "Player wins".

Aim: 2. Create a 3D application where AI chases a Player
Step 1: Setup New Scene
* Go to File > New Scene (Ctrl+N) and select Basic (Built-in).
* Create a Plane and two Capsules named Player and Enemy.
* Output Description: The scene contains a grey floor with two grey upright capsules.
Step 2: Movement Script (Player1.cs)
Attach this script to the Player capsule.
C#
using UnityEngine;


public class Player1 : MonoBehaviour
{
    public float speed = 5.0f;


    void Update()
    {
        float moveHorizontal = Input.GetAxis("Horizontal");
        float moveVertical = Input.GetAxis("Vertical");
        Vector3 movement = new Vector3(moveHorizontal, 0.0f, moveVertical);
        transform.Translate(movement * speed * Time.deltaTime, Space.World);
    }
}


Step 3: AI Chasing Script (AI_chasePlayer.cs)
Attach this script to the Enemy capsule.
C#
using UnityEngine;
using UnityEngine.AI;


public class AI_chasePlayer : MonoBehaviour
{
    public Transform player;
    public float detectionRange = 4.0f;
    private NavMeshAgent agent;


    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
    }


    void Update()
    {
        float playerDistance = Vector3.Distance(transform.position, player.position);
        
        if (playerDistance < detectionRange)
        {
            agent.SetDestination(player.position);
            Debug.Log("AI is chasing the player");
            Debug.Log(playerDistance);


            if (playerDistance < 0.5f)
            {
                Debug.Log("Enemy wins!");
                StopGame();
            }
        }
        else
        {
            agent.ResetPath();
        }
    }


    void StopGame()
    {
        Debug.Log("Game over! The enemy caught you.");
        Application.Quit();
        UnityEditor.EditorApplication.isPlaying = false;
    }
}


* Action: In the Enemy Inspector, drag the Player object into the Player slot.
Step 4: NavMesh and RigidBody
* Player: Add Rigidbody, uncheck Use Gravity, and Freeze Rotation for X, Y, Z.
* Enemy: Add NavMesh Agent component.
* Baking: Create an empty object, add NavMesh Surface, and click Bake.
Step 5: Final Execution
* Press Play.
* Output Description:
   * The Enemy remains stationary until the Player moves within the Detection Range.
   * Once close, the Enemy follows the Player's movements.
   * If the Enemy touches the Player, the Console prints "Enemy wins!" and "Game over! The enemy caught you.".
