Practical 5: Implement 2D Effectors
Aim: To implement different 2D Effectors in Unity, including Surface Effector, Platform Effector, Area Effector, and Buoyancy Effector.
________________


A. Surface Effector
The Surface Effector 2D applies forces along the surface of a collider, effectively creating conveyor belts or friction-modified surfaces.
Step 1: Create a New Scene
* Open Unity and create a new scene.
Step 2: Create Player Object
* Right-click in the Hierarchy panel.
* Select 2D Object → Sprite → Square.
* Rename the object to PlayerObject.
* In the Inspector panel, set the Transform Position to:
   * X=0
   * Y=2
   * Z=0
Step 3: Add Physics Components to Player
1. With PlayerObject selected, go to the Top Menu Bar.
2. Navigate to Component → Physics 2D → Rigidbody 2D.
3. In the Rigidbody 2D settings, set the Gravity Scale to 3.
4. Again, go to Component → Physics 2D → Box Collider 2D to add a collider to the player.
Step 4: Create Surface Object
1. Right-click in the Hierarchy.
2. Select 2D Object → Sprite → Square.
3. Rename this object to Surface.
4. In the Inspector, set the Transform Scale to:
   * X=6
   * Y=0.5
   * Z=1
5. Set the Transform Position to:
   * X=0
   * Y=-2
   * Z=0
Step 5: Add Surface Effector 2D
1. Select the Surface object.
2. Go to the Top Menu: Component → Physics 2D → Box Collider 2D.
3. In the Box Collider 2D component in the Inspector, tick the Used By Effector checkbox.
4. Go to the Top Menu: Component → Physics 2D → Surface Effector 2D.
5. In the Surface Effector 2D component, set the following values:
   * Speed = 3
   * Force Scale = 0
Output Description: The scene shows a white square (Player) positioned above a long white rectangular platform (Surface). When played, the player falls onto the surface, and the Surface Effector will move the player along its length at the specified speed.
________________


B. Platform Effector
The Platform Effector 2D allows for "one-way" collisions, where objects can pass through the platform from one side (usually the bottom) but land on it from the other.
Step 1: Create Platform Object
* In the Hierarchy panel, Right-click → 2D Object → Sprite → Square.
* Rename the object to Platform.
Step 2: Add Box Collider 2D to Platform
1. With Platform selected, go to Top Menu → Component → Physics 2D → Box Collider 2D.
2. In the Inspector for Box Collider 2D:
   * Tick Used by Effector.
   * Ensure Is Trigger is unchecked.
Step 3: Add Platform Effector 2D
1. With Platform selected, go to Top Menu → Component → Physics 2D → Platform Effector 2D.
2. In the Platform Effector 2D settings, tick Use One Way.
3. Ensure the Surface Arc is set to 180.
Step 4: Add Player Control Script
1. Click on the PlayerObject.
2. Click Add Component and create a new script named plat.
3. Enter the following code:
C#
using UnityEngine;


public class plat : MonoBehaviour
{
    public float speed = 5f;
    public float jumpForce = 8f;
    private Rigidbody2D rb;


    void Start()
    {
        rb = GetComponent<Rigidbody2D>();
    }


    void Update()
    {
        float move = Input.GetAxis("Horizontal");
        rb.velocity = new Vector2(move * speed, rb.velocity.y);


        // Jump
        if (Input.GetKeyDown(KeyCode.Space))
        {
            rb.AddForce(Vector2.up * jumpForce, ForceMode2D.Impulse);
        }
    }
}


Output Description: The scene contains the PlayerObject between a Surface Effector platform and a Platform Effector platform. Using the side arrow keys allows left/right movement, and the spacebar triggers a jump. The player can jump through the Platform from below and land on its top surface.

C. Area Effector
The Area Effector 2D applies forces within a defined area when a collider enters that space.
Step 1: Create Effector Area
1. In Hierarchy → Right-click → 2D Object → Sprite → Square.
2. Rename the object to Area.
3. In the Transform, set the position and scale as follows:
   * Position: X=6.34, Y=0.8971, Z=0
   * Scale: X=7, Y=1.6657, Z=1
Step 2: Add Box Collider 2D
1. With the Area object selected, go to Component → Physics 2D → Box Collider 2D.
2. Tick Is Trigger.
3. Tick Used By Effector.
Step 3: Add Area Effector 2D
1. With Area selected, go to Top Menu → Component → Physics 2D → Area Effector 2D.
Step 4: Add Area Script
Apply a script named area to the player with the following code to handle movement through the area:
C#
using UnityEngine;


public class area : MonoBehaviour
{
    [Header("Movement Settings")]
    public float speed = 5f; // Horizontal speed
    public float jumpForce = 8f; // Jump strength
    private Rigidbody2D rb;


    void Start()
    {
        // Get Rigidbody2D component attached to player
        rb = GetComponent<Rigidbody2D>();
    }


    void Update()
    {
        // Horizontal movement
        float move = Input.GetAxis("Horizontal"); // -1 left, 1 right
        rb.velocity = new Vector2(move * speed, rb.velocity.y);


        // Jump movement
        if (Input.GetKeyDown(KeyCode.Space))
        {
            rb.AddForce(Vector2.up * jumpForce, ForceMode2D.Impulse);
        }
    }
}


Output Description: The scene shows the player object placed between the Area Effector zone and the surface. The player will be able to pass through this specific designated area while horizontal forces or other area-specific physics are applied.

D. Buoyancy Effector
The Buoyancy Effector 2D creates simple fluid behavior like floating or resistance.
Step 1: Create Water Surface
1. In Hierarchy → Right-click → 2D Object → Sprite → Square.
2. Rename the object to water.
Step 2: Add Box Collider 2D
1. With water selected, go to Inspector → Add Component → Physics 2D → Box Collider 2D.
2. Tick Is Trigger (Required for the effector to detect objects).
3. Tick Used By Effector.
Step 3: Add Buoyancy Effector 2D
1. With water selected, go to Top Menu → Component → Physics 2D → Buoyancy Effector 2D.
2. In the Inspector, set the following values:
   * Density = 3
   * Surface Level = 0.5
Output Description: The scene view shows a white rectangular area representing water. When the player enters this area, they will initially float and eventually settle on the "water" surface due to the buoyancy settings.
________________


E. Audio Source
Adding sound effects or background music to the scene.
1. Navigate to Component → Audio → Audio Source.
2. In Audio Resource, select the desired clip (e.g., "jethalal_b").
3. Configure settings in the Inspector:
   * Play On Awake: Checked
   * Priority: 128
   * Volume: 1
   * Pitch: 1
